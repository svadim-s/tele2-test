# Описание проекта

Веб приложение по предоставлению различных товаров.

Реализовано:
- пагинация по списку продуктов
- поиск по названию продукта.

Использован официальный API https://dummyjson.com/docs/products

# Запуск проекта

```
yarn install
yarn start
```

# Stack

- React
- Redux Toolkit
- TypeScript
- Vite
- SCSS
